# delm
Elm style UI library for desktop applications.

## Features
* Simple - inspired by [elm](https://elm-lang.org/) providing simple architecture that is easy to use.
* Multiple rendering APIs - delm uses [wgpu](https://wgpu.rs/) providing support for Vulkan, DirectX, Metal, OpenGL and even WebGL

## Future plans
* DOM backend - Using DOM based rendering to allow building web applications in the future.
